# AWS-EMR-setup

Installing pip3:

```bash
sudo yum install python34-setuptools
sudo easy_install-3.4 pip
```
Using python3 with pyspark:
```bash
export PYSPARK_PYTHON=python3
```
Install findspark:
```bash
sudo /usr/bin/pip3 install findspark
```
